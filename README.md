Система бронювання квитків у кінотеатрі

Проект реалізує систему бронювання квитків у кінотеатрі з використанням об'єктно-орієнтованого програмування (ООП) та шаблонів проектування на мові Java.

Функціонал

- **Управління сеансами**: Створення, перевірка на накладання часу
- **Бронювання квитків**: Виділення місць, перевірка доступності
- **Типи квитків**: Standard та VIP з різними цінами
- **Система оплати**: Підтримка різних способів оплати (готівка, карта)
- **Обробка помилок**: Власна ієрархія виключень
- **Тестування**: Повне покриття основного функціоналу

Архітектура

Проект побудований з використанням наступних шаблонів проектування:

Породжувальні шаблони
- **Singleton**: `CinemaDatabase` - єдина точка доступу до даних
- **Factory Method**: `TicketFactory` - створення різних типів квитків

Структурні та поведінкові шаблони
- **Strategy**: `PaymentStrategy` - різні способи оплати
- **Facade**: `BookingService` - спрощений інтерфейс для бронювання
- **Iterator**: Перебір колекцій сеансів
- **Null Object**: Захист від NullPointerException

Принципи GRASP
- **Controller**: `BookingService` координує процес бронювання
- **Information Expert**: Класи мають інформацію для виконання своїх завдань
- **Creator**: `ScheduleService` відповідає за створення сеансів
